Gulp HTML5 Mobile/Web Game

Gulp is a mobile-browser game made in HTML5 back when mobile browsers first started supporting HTML5 and hardware acceleration via WebGL. I was excited about HTML5 and already had a lot of experience with game development, so naturally I dove right into HTML5 game development. This game was my learning project for HTML5, and I also gained a much better understanding of the workings of javascript, and specifically its prototypal OOP, while making it. Getting the game working was easy, the main challenge was optimizing it to the point where it would run at a decent framerate on mobile devices; at the time this game was made realtime mobile-browser games were extremely simple due to performance restrictions. The game will render using WebGL to take advantage of hardware-acceleration on compatible devices, but falls back on software rendering on devices that aren't compatible with WebGL. This version of the game has all the core gameplay intact but details such as instructions and a scoreboard are unpolished. A couple other very similar versions of the game were made to accomodate mobile web game portal websites who bought licenses to host this game on their site - similar to flash games in the early days of the flash game portals, even the simplest mobile HTML5 games sold for a decent price back when I made this.

Android Math Game

This game is another Android app created for a client when I was freelancing. The gameplay is simple, it involves placing weights on a balance to try and even it out by getting the fractions on the weights on each side of the scale to add up to the same amount. The physics in the game are real, instead of simply using animations to make the balance tip to whichever side had more weight, it is created from several pieces, connected at a pivot, and the simulated mass of each of the weights is actually equal to the fraction written on it. The physics simulation is powered by Box2D, which gives the game a more realistic feel and provides intuitive feedback for the player.

This app isn't on any app store, as far as I understand this app was created for the client to use with their own students/school. Along with a 5-star review, the client left the following feedback on freelancing site elance.com:

"We are glad to work with him. We are really satisfied with his work and his report."

Tech: Javascript, HTML5, Phaser
